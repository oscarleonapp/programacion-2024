En HTML, las cosas giran en torno a las etiquetas. Las etiquetas comienzan con <, seguidas de una palabra clave y termina con >.

!Creemos nuestra primera etiqueta <button> en el archivo index.html a continuación!

¿Cuál de estas etiquetas HTML crea un botón?

<button> es una etiqueta de apertura. Le dice al navegadoor web que queremos comenzar a mostrar un botón

¿Qué crees que viene después de la etiqueta de apertura <button>?

La mayoría de las etiquetas en HTML vienen en pares, tienen una etiqueta de apertura y una etiqueta de cierre. Lo que codificó anteriormente es la etiqueta de apertura de un elemento botón.

Vamos a codficar la etiqueta de cierre del botón agregando </button> después de la etiqueta del botón de apertura.
